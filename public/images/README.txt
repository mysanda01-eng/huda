IMAGES - Upload Here
====================

For "The Way I See You" gallery section:
- You need 7 images total
- File format: .jpg, .png, .webp (any common image format)
- Recommended size: 600x600px or larger (square for best layout)
- Naming: You can name them anything (e.g., image1.jpg, photo1.jpg, etc.)

Optional: Add captions
- In the HTML file, you can add a short sentence under each image
- Edit index.html and look for the "image-caption" class sections

Images will display in a responsive grid layout, automatically adapting to screen size.
